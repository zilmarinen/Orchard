# Introduction
Orchard is a macOS application designed to facilitate the rapid development of 3D games on both macOS and tv/iOS. Written collaboratively alongside [Harvest](https://github.com/zilmarinen/Harvest) and [Meadow](https://github.com/zilmarinen/Meadow), Orchard is a 2D modelling application used to design, visualise and export 3D scene graphs.

## Dependencies
[Euclid](https://github.com/nicklockwood/Euclid) is a Swift library for creating and manipulating 3D geometry and is used extensively within this project for mesh generation and vector operations.
